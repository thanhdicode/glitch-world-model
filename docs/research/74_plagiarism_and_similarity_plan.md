# Plagiarism and Similarity Plan

Date: 2026-06-18

Status: active integrity plan

## Drafting Rules

1. Draft from repository evidence and verified primary sources only.
2. Do not paste long language from source papers into the manuscript.
3. Paraphrase method and benchmark context in fresh prose after reading the source.
4. Keep direct quotations out of the paper unless a later explicit need arises.
5. All manuscript text must be original prose written for this paper.
6. Reused project text from repository notes must be rewritten and cited or linked through the
   source matrix when necessary.
7. Do not copy paper wording from cited sources.

## Citation Rules

1. Every factual benchmark or method statement should point to a source in `paper/references.bib`
   or a verified repository document.
2. Do not add placeholder citations.
3. Do not cite papers that have not been checked at the primary-source level during the later
   bibliography pass.

## Similarity Control Workflow

1. Finish section prose from the claim map.
2. Run a manual similarity review against nearby source notes and the cited papers.
3. Recheck tables, captions, and appendix text for copied phrasing.
4. Run formal similarity screening before submission.
5. If any direct quote is unavoidable, keep it minimal and cite it clearly; direct quotes should
   generally be avoided in the paper.
6. Freeze the submission package only after the source matrix and final bibliography agree.

## Known Current Limitation

The manuscript is still at scaffold level, so a later final-prose pass remains necessary before any
formal similarity screening.
