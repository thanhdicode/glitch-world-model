## Notes

- `GlitchBench` remains benchmark-relevant because the public Hugging Face artifact is online and structured, but the repo should treat it as image-only evidence.
- `TempGlitch` is the best methodological fit for the project hypothesis, yet it is still a paper-first benchmark until the public access path is verified in Phase 2.
- `VideoGlitchBench` is the richest temporal benchmark on paper, but it is not yet the safest operational fallback because public release details were not verified.
