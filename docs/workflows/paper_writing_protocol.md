# Paper Writing Protocol

- Build tables from documented summaries or explicitly supplied artifacts.
- Every result table identifies split and provenance.
- Keep validation-only and locked-test metrics visibly separate.
- Cite primary sources where possible.
- Register each paper-facing claim before publication.
- State negative results, dataset limitations, heuristic grouping, and prior subset exposure.
- Do not write LeWM, JEPA, SIGReg, SOTA, or neural locked-test claims before their gates pass.

Use `python scripts/make_paper_tables.py` and check `paper/README.md` for build commands.
