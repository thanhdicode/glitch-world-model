# Paper Claim Rules

- Register every paper-facing claim before publication.
- A LeWM-framed title or contribution requires Gate 7: real checkpoint surprise scores and
  finite gameplay validation metrics with recorded hashes.
- A neural locked-test metric requires Gate 10 and an explicit one-time release.
- Never claim state of the art without broad, protocol-compatible benchmark comparison.
- Never claim temporal localization without verified temporal spans and temporal metrics.
- Never claim real-time operation without reproducible FPS/latency measurement.
- Keep validation-only, diagnostic/exposed-test, and locked-test results visibly separate.
- Name zero-action or action-free adaptations explicitly; do not present them as unchanged
  action-conditioned LeWM.
- Keep negative results, heuristic grouping, binary-label limits, and prior exposure visible.
- Use primary sources and record source/access details.

The current paper now targets Springer LNICST through `llncs`, but the repository still lacks the
official Springer class files and local TeX toolchain. Treat format-readiness as blocked until an
official-kit compile confirms bibliography, page fit, warnings, and accessibility requirements.
