\section{Submission Checklist}
\label{app:submission-checklist}

Before submission, verify the following:
\begin{itemize}[leftmargin=*]
\item compile the manuscript with the official Springer \texttt{llncs} template files;
\item replace placeholder author and institute metadata with final approved metadata;
\item confirm that the abstract, introduction, and conclusion remain within the claim map;
\item verify that every table entry and every citation resolves to a checked repository source or
  primary reference;
\item ensure that no WOB result, locked-test result, or unsupported superiority language has been
  introduced;
\item run the repository validation suite and retain the exact outputs in the submission notes.
\end{itemize}
