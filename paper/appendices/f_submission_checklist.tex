\section{Submission Checklist}
\label{app:submission-checklist}

Before submission:
\begin{itemize}[leftmargin=*]
\item compile with the official Springer class and bibliography files;
\item confirm anonymous-submission metadata for review and prepare camera-ready metadata separately;
\item audit the abstract, all results paragraphs, and conclusion against claims C-090--C-092,
C-095, C-098, C-100--C-102, C-110--C-115, and the corresponding forbidden-claim notes;
\item verify every table value against its checked research report;
\item upload only provenance-recorded, publication-ready figures; do not upload local planning docs
or raw output directories;
\item confirm page count, references, captions, accessibility text, and similarity screening;
\item ensure no generated evidence, checkpoint, dataset, cache, or credential is staged; and
\item rerun the complete repository validation suite.
\end{itemize}
