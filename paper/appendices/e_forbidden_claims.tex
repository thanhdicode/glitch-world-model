\section{Forbidden Claims for the Current Draft}
\label{app:forbidden-claims}

The current manuscript phase must not introduce any of the following as positive claims:
\begin{itemize}[leftmargin=*]
\item state-of-the-art performance;
\item broad superiority over alternative methods;
\item locked-test performance;
\item World of Bugs training or evaluation performance;
\item SIGReg-specific benefit;
\item real-time capability;
\item temporal localization, unless directly supported by later validated evidence;
\item generalization beyond the current non-locked TempGlitch validation-only family.
\end{itemize}

This appendix is intentionally redundant with the claim map because the easiest paper mistake is to
forget which statements were still out of bounds.
