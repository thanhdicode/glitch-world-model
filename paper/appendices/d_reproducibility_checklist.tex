\section{Reproducibility Checklist}
\label{app:reproducibility-checklist}

The current scaffold follows checklist-style reproducibility discipline motivated by prior ML
reporting work \cite{pineau2021reproducibility}. It satisfies or tracks the following repository
controls:
\begin{itemize}[leftmargin=*]
\item artifact-backed checkpoint identities for seed42, seed43, and seed44;
\item frozen manifest and output hashes for the R5 identical-episode family;
\item explicit threshold policy and metric definitions;
\item an appendix-level claim boundary and forbidden-claims list;
\item repository control documents for source mapping, gap logging, and submission packaging.
\end{itemize}

Remaining reproducibility work includes bibliography expansion from verified primary sources,
final venue template validation with the official Springer files, and figure-level audit before any
camera-ready export.
