\section{Problem Formulation}
\label{sec:problem-formulation}

Let episode $e$ contain an ordered set of video windows
$\mathcal{W}_e=\{w_{e,1},\ldots,w_{e,T_e}\}$ and a binary episode label
$y_e\in\{0,1\}$, where 0 denotes normal and 1 denotes buggy. A scorer $s$ assigns a scalar anomaly
value $s(w_{e,t})$ to each window. An aggregation function $A$ produces one episode score
\begin{equation}
S_e=A\left(s(w_{e,1}),\ldots,s(w_{e,T_e})\right).
\end{equation}
The recorded aggregation family includes mean, maximum, and the mean of the two largest window
scores. Window values are diagnostic intermediates; episode scores are the reporting unit.

For thresholded metrics, a threshold $\tau$ is computed exclusively from designated normal
calibration episodes. Predictions are $\hat{y}_e=\mathbb{1}[S_e\geq\tau]$. Ranking metrics
(AUROC and AUPRC) and threshold metrics (F1, precision, recall, balanced accuracy) are reported
together with FPR@95TPR. This combination exposes both class separation and the false-positive
cost near high recall.

The question is deliberately narrow: on a frozen non-locked evaluation split, do recorded latent
surprise configurations show useful same-support separation between normal and buggy episodes
relative to named simple baselines? The formulation does not test temporal localization,
deployment robustness, or generalization to an unspecified population of games.
